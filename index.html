Robotics

Mostly simple robotic works. So far, I've done robot projects using Arduino, NodeMCU, and Raspberry Pi.

Beach cleaner robot

A small mobile robot that will send you live video so you can control it. And It will pick up trash on your command. All of this can be controlled on your phone using WIFI and Blynk application.

Automatic bin

A modified robot from a circle -lid trash can. Operated by IR sensor that will detect objects close to the lid and uses a Servo motor to rotate the lid. It's a simple yet useful mechanic-based robot.

Automatic cat feeder

This is the first robot that I made. Most of the components are from used box, which is eco-friendly robot. It uses Arduino UNO as a timer and power adapter and a Servo motor as a food gateway.

Automatic fish feeder

Using the same idea as the cat-feeder with more experiences and better components, this one is far better than the first one I made. It works by twirling the long metal inside the tube, using 9V motor. the length and timing can be adjusted by a timer.
